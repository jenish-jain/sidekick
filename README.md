# Your own sidekick

### how to run it locally
> get random dad joke 
`` go run main.go dadjoke``


### how to use the tool from homebrew
````shell
brew tap jenish-jain/tap
brew install jenish-jain/tap/sidekick

sidekick dadjoke
````
